# 📊 Customer Retention Dashboard

## 🚀 Live App
https://customer-retention-project-d56eye4k7enbfgbzwqgmvo.streamlit.app/

## 📌 Project Overview
This project analyzes customer churn behavior using an interactive dashboard built with Streamlit.

## 🎯 Objectives
- Analyze customer churn patterns
- Identify high-value customers at risk
- Understand impact of engagement and product usage
- Build a retention scoring model

## 🛠️ Tools & Technologies
- Python
- Pandas
- Streamlit

## 📊 Key Features
- Interactive filters (Active / Inactive customers)
- KPI metrics (Total customers, churn rate, average balance)
- Engagement vs churn analysis
- Product usage vs churn analysis
- High-value at-risk customer detection
- Retention scoring model

## 📌 Key Insights
- Active customers have lower churn rates
- Customers using more products are more loyal
- High-balance inactive customers are at high risk
- Retention scoring helps identify loyal vs risky customers
